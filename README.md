## Getting Started

This is the repo for the skyrim dataset experiment

### Prerequisites

- PyTorch ~0.4.0
- torchvision ~0.3.0
- numpy
- PIL
- tqdm

### Installation

```sh
git clone https://github.com/IvoryCandy/experience-extraction-from-gameplay-video.git
```

### Usage`

run

```sh
python3 main.py
```
